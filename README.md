# Honda Cars Dasmariñas

This project is an ongoing update for the **Honda Cars Dasmariñas** website.

## Project Files

- `index.html` – main landing page
- `maison-templatemo.html` – additional page/template file
- `templatemo-maison-style.css` – site styling
- `templatemo-maison-doree.js` – interactive behavior/scripts
- `images/` – image assets

## Run Locally

1. Download or clone this project.
2. Open `index.html` in your browser.
